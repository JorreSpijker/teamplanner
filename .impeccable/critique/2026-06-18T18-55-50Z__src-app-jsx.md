### [P1] No undo and no confirmation for destructive actions
Three paths destroy work with zero recovery: × on TeamCard (immediate), delete player (immediate), new-plan import (wipes all teams).

Command: /impeccable harden

### [P1] Two accent colors violate One Accent Rule

Fix: Make export button neutral style (bg-white border), keep blue-600 as sole primary accent.

Jeugd/B team: category + age + bandwidth + format + gender count + color rule = 6 equal-weight chips. Color rule chip is very wide. Eye has nowhere to land.
